# SpeakEasy — Zeabur Deployment Guide
# SpeakEasy — Zeabur 部署指南

## Why Zeabur? 為什麼用 Zeabur？

| Feature 特點 | Benefit 好處 |
|-------------|-------------|
| HTTPS | Voice API requires HTTPS. Zeabur provides it free. 語音 API 需要 HTTPS |
| Custom Domain | Can use your own domain 可以綁自己的域名 |
| Always On | 24/7 available, not just when your PC is on 隨時可用 |
| Samsung Phone | Android Chrome needs HTTPS for microphone 安卓 Chrome 麥克風需要 HTTPS |

**Important 重要**: Web Speech API (microphone 麥克風) only works on:
- `localhost` (local testing 本地測試)
- HTTPS sites (Zeabur provides this 部署後自動有)
- NOT on `http://` (microphone will be blocked 麥克風會被封鎖)

So deploying to Zeabur is **required** for mobile voice input to work!
所以部署到 Zeabur 是手機語音輸入的**必要條件**！

## Method: Static Site on Zeabur
## 方法：Zeabur 靜態網站

### Step 1: Create GitHub Repo 建立 GitHub Repo

```bash
# In the app folder 在 App 資料夾內
cd "📥 收件匣/2026-04-03_SpeakEasy英語口說App"

# Init git
git init
git add .
git commit -m "SpeakEasy v1.0 - AI English Coach"

# Create GitHub repo (using Claude Code's gh command)
gh repo create speakeasy-app --public --source=. --push
```

### Step 2: Deploy on Zeabur 在 Zeabur 部署

1. Go to 前往 → [dash.zeabur.com](https://dash.zeabur.com)
2. Click **"New Project"** 點「新專案」
3. Click **"Deploy New Service"** → **"Git Repository"**
4. Select your **speakeasy-app** repo 選你的 repo
5. Zeabur auto-detects it as a static site 自動偵測為靜態網站
6. Wait for deployment (< 1 minute) 等待部署（不到一分鐘）
7. Click **"Networking"** → **"Generate Domain"** 產生域名
8. You get a URL like: `speakeasy-app.zeabur.app` ✅

### Step 3: Open on Samsung Phone 在三星手機開啟

1. Open Chrome on your Samsung 在三星手機開 Chrome
2. Go to `https://speakeasy-app.zeabur.app` 打開網址
3. Chrome Menu → **"Add to Home Screen"** 新增至主畫面
4. Now it looks and works like a native app! 現在跟原生 App 一樣！

## File Structure for Zeabur
## Zeabur 檔案結構

Zeabur serves static files as-is. No changes needed.
Zeabur 直接 serve 靜態檔案，不需要改任何東西。

```
/ (root)
├── SpeakEasy_2026-04-03.html   → rename to index.html for cleaner URL
├── manifest.json
├── service-worker.js
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

**Tip 提示**: Rename the HTML file to `index.html` before deploying so the URL is just `https://your-domain.zeabur.app/` without the filename.
部署前把 HTML 檔改名為 `index.html`，URL 就不用帶檔名。

## Optional: Custom Domain 選用：自訂域名

If you have a domain 如果你有自己的域名:
1. In Zeabur dashboard → Networking → Custom Domain
2. Add your domain (e.g. `english.yoursite.com`)
3. Add CNAME record in your DNS: `your-app.zeabur.app`
