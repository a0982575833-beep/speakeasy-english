# SpeakEasy — AI 英語口說教練

## Quick Start 快速開始

### Step 1: Open the App 打開 App
Double-click `SpeakEasy_2026-04-03.html` in your browser.
用瀏覽器打開 `SpeakEasy_2026-04-03.html`。

### Step 2: Enter API Key 輸入 API Key
Your Google Gemini API Key is already in `APIKEY.txt` on your Desktop.
你的 Google Gemini API Key 已經在桌面的 `APIKEY.txt` 裡。

Paste it into the setup screen and click "Start Learning".
貼到設定畫面，點「Start Learning」。

### Step 3 (Mobile): Add to Home Screen 加到手機桌面

**iPhone (Safari):**
1. Open the HTML file URL in Safari 在 Safari 打開
2. Tap the Share button (square with arrow) 點分享按鈕
3. Tap "Add to Home Screen" 點「加到主畫面」
4. Tap "Add" 點「加入」

**Android (Chrome):**
1. Open in Chrome 在 Chrome 打開
2. Tap the three dots menu 點三個點選單
3. Tap "Add to Home screen" 點「新增至主畫面」
4. Tap "Add" 點「新增」

> Tip: To use on phone, you need to serve the file via a local server or upload to a hosting service. The simplest way is to use VS Code Live Server or Python: `python -m http.server 8080`
> 提示：手機使用需要透過本地伺服器或上傳到 hosting 服務。最簡單的方法是用 VS Code Live Server 或 Python: `python -m http.server 8080`

## How to Get a Free Gemini API Key 如何取得免費金鑰

1. Go to 前往 → https://aistudio.google.com/apikey
2. Sign in with your Google account 用 Google 帳號登入
3. Click **"Create API Key"** 點擊「建立 API 金鑰」
4. Select **"Create API key in new project"** 選「在新專案中建立」
5. Copy the key 複製金鑰 (starts with `AIza...`)
6. Paste into SpeakEasy 貼到 SpeakEasy App

**Free tier limits 免費額度:**
- 15 requests per minute 每分鐘 15 次請求
- 1 million tokens per minute 每分鐘 100 萬 tokens
- More than enough for personal practice! 個人練習綽綽有餘！

## Features 功能說明

### 1. AI Chat 對話 (💬)
Chat with NingNing in English. She'll correct your mistakes and teach you better expressions.
跟 NingNing 用英文聊天。她會糾正你的錯誤，教你更好的表達方式。

- **Voice input 語音輸入**: Hold the 🎤 button and speak English
- **Text input 文字輸入**: Type when you can't speak
- **Gentle/Strict mode**: Toggle between correcting major errors only vs. everything

### 2. Daily Scenarios 每日情境 (📋)
3 new practice scenarios every day, tailored to your work and life.
每天 3 個新練習情境，針對你的工作和生活量身定制。

Categories 分類:
- 🏢 Work 工作: Boss reports, client calls, emails
- 🌍 Social 社交: Meeting friends, networking, small talk
- ☕ Daily 日常: Ordering food, shopping, travel

### 3. Shadowing 跟讀 (🎤)
NingNing says a sentence → You repeat → Compare your attempt.
NingNing 說一句 → 你跟著說 → 比較你的嘗試。

Best way to improve pronunciation and rhythm!
改善發音和語感最有效的方法！

### 4. Vocabulary 生詞本 (📖)
- **Words 單字**: Save words from conversations
- **Phrases 句庫**: Save full reusable sentences
- **Review 複習**: Flashcard-style review

### 5. Stats 統計 (📊)
- Daily streak 連續天數
- XP points 經驗值
- Level progression 等級進度
- Common mistakes tracking 常犯錯誤追蹤

## Speed Control 語速控制

- **Slow 慢速** (0.7x): For beginners, hear every word clearly 初學者用，每個字都聽清楚
- **Normal 正常** (1.0x): Natural conversation speed 自然對話速度
- **Fast 快速** (1.3x): Challenge yourself! 挑戰自己！

## Tips for Best Results 最佳使用建議

1. **Practice daily 每天練習** — Even 5 minutes helps! 即使 5 分鐘也有幫助！
2. **Use voice 多用語音** — Speaking is the goal, not typing 口說才是目標
3. **Don't be afraid of mistakes 不要怕犯錯** — NingNing is patient and encouraging
4. **Start with Shadowing 從跟讀開始** — Best warm-up before chatting 聊天前最好的暖身
5. **Review saved words 複習存過的字** — Use flashcards weekly 每週用閃卡複習

## Troubleshooting 疑難排解

**"Voice not supported" 語音不支援:**
- Use Chrome or Safari (Firefox has limited support)
- 使用 Chrome 或 Safari（Firefox 支援有限）

**"API error" API 錯誤:**
- Check your API key is correct 確認 API 金鑰正確
- Check internet connection 確認網路連線
- Free tier may have rate limits 免費額度可能有速率限制

**No sound 沒有聲音:**
- Check device volume 確認裝置音量
- Some browsers need user interaction before playing audio
- 有些瀏覽器需要使用者互動後才能播放音訊
